Quem Somos

Hortfruit Online é uma loja virtual (e-commerce) fictícia criada como Projeto Extencionalista "Desenvolvimento de uma Loja Virtual com Bootstrap 5 e Node.js" disponibilizado gratuitamente por Yale Designer.

O código-fonte pode ser obtido via GitHub. Este projeto pode ser utilizado livremente como base para outras lojas virtuais, sem ônus para o usuário.
